# Sound Effects Folder

Add your Doctor Strange sound files here for the full experience!

## Required Files:

- **doctor_strange_theme.mp3** - Background music (Doctor Strange main theme)
- **portal_open.wav** - Portal opening sound effect (optional)
- **throw.wav** - Energy disc throw sound effect (optional)
- **gesture.wav** - Hand gesture sound effect (optional)

## Where to Find Sounds:

You can download Doctor Strange sound effects and theme music from:
- YouTube (search "Doctor Strange theme" or "Doctor Strange sound effects")
- Free sound effect websites
- Movie soundtracks

## Note:

The application will run without sounds if this folder is empty. Sound effects are optional enhancements.
